# XBOX
Dashboard de Vendas do Xbox com Excel (LibreOffice Calc)

Perspectiva lógica de análise e não de fórmulas.

Criar um Dashboard de Vendas do Xbox para responder perguntas de negócio:

    • Quanto foi faturado?
    • Qual plano gera mais dinheiro?
    • Como estão distribuídas as assinaturas?
    • Existem padrões claros de consumo? 
    
Base

    • Cada linha representa um assinante
    • Cada coluna representa uma característica desse assinante
    • Total Value já é o valor final pago
    • Plan define o tipo de assinatura
    
Agrupar os assinantes por Plan e somar os Total Value, tem-se o faturamento por plano.

O que o dashboard precisa mostrar:

Indicadores principais (KPIs)

    • Faturamento total
    • Faturamento por plano
    • Quantidade de assinantes por plano
    
Esses números contam a história principal.

“Leitura” das “abas”:

Aba 1 — Base

    • Dados crus
Aba 2 — Cálculos

    • Totais
    • Agrupamentos
    • Consolidação por plano
Aba 3 — Dashboard

    • Gráficos
    • Comparações visuais
    • Leitura rápida
    
Lógica do faturamento por plano:

Raciocínio:

    1. Olhar todos os assinantes
    
    2. Separar:
        ◦ Ultimate
        ◦ Core
        ◦ Standard
        
    3. Para cada grupo:
        ◦ Considerar apenas o Total Value
        
    4. Comparar os totais
    
Gráficos

O que devem responder:

    • Comparar planos entre si
    • Proporção de cada plano no faturamento total
    • Quantidade de clientes por plano
